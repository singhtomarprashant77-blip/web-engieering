# 🔐 Web Engineering Authentication System

A responsive authentication system built with HTML, CSS, JavaScript and PHP.

## 🚀 Features
- User Registration
- User Login
- Forgot Password
- Reset Password
- OTP Email Verification

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL (if used)

## 📂 How to Run
1. Install XAMPP
2. Move project folder to htdocs
3. Start Apache & MySQL
4. Open browser: http://localhost/your-folder-name
